# 执行步骤 — 天堂次元港

## 推进原则

- 每次只执行一步，完成后暂停，等待确认
- 每一步完成后记录开发日志到 `devlogs/YYYY-MM-DD.md`
- 每步验证：`npm run dev` → 浏览器测试 → 确认通过
