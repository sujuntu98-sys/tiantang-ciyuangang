# 需求文档 — 天堂次元港

## 项目名称

天堂次元港（Tengoku Jigen-kou / Heaven Dimension Port）

## 项目定位

面向二次元爱好者的图片分享社区网站。

## 目标用户

- 二次元动漫爱好者
- 画师 / 同人创作者
- ACG 内容消费者

---

## 功能需求

### 1. 账号系统

| 编号 | 功能 | 说明 |
|------|------|------|
| A1 | 用户注册 | 用户名 + 邮箱 + 密码注册 |
| A2 | 用户登录 | 邮箱 + 密码登录 |
| A3 | 个人主页 | 展示用户信息、发布的帖子 |
| A4 | 个人设置 | 修改头像、个人简介 |
| A5 | 登出 | 安全退出登录 |

### 2. 发帖系统

| 编号 | 功能 | 说明 |
|------|------|------|
| P1 | 图片上传 | 用户上传二次元角色图片 |
| P2 | 帖子发布 | 标题 + 描述 + 图片 |
| P3 | 类型标记 | 标记为「原画」或「二创」 |
| P4 | 标签系统 | 用户可添加标签（作品名、角色名等） |

### 3. 互动系统

| 编号 | 功能 | 说明 |
|------|------|------|
| I1 | 点赞 | 用户可对帖子点赞/取消点赞 |
| I2 | 收藏 | 用户可收藏/取消收藏帖子 |
| I3 | 评论 | 用户可在帖子下发表评论 |
| I4 | 关注 | 用户可关注/取关其他用户 |

### 4. 内容展示

| 编号 | 功能 | 说明 |
|------|------|------|
| C1 | 首页最新 | 按时间倒序展示全站帖子 |
| C2 | 首页热门 | 按点赞数排序展示热门帖子 |
| C3 | 帖子详情 | 大图展示 + 完整信息 + 评论区 |
| C4 | 搜索 | 按标签/关键词搜索帖子 |

---

## 非功能需求

- **响应式设计**：适配桌面端和移动端
- **图片安全**：限制上传文件类型（jpg, png, webp）和大小（最大 10MB）
- **数据安全**：用户密码加密存储，API 路由鉴权保护
- **浅色清新风格**：以白/浅蓝/浅粉为主色调
