# 设计规范 — 天堂次元港

## 整体风格

浅色清新风，以白/浅蓝/浅粉为主色调，参考 bilibili 的清爽感。

## 配色方案

| 用途 | 颜色 | TailwindCSS |
|------|------|-------------|
| 主背景 | 白色 | `bg-white` / `bg-gray-50` |
| 主色调 | 天蓝 | `blue-500` (#3B82F6) |
| 辅助色 | 樱花粉 | `pink-400` (#F472B6) |
| 强调色 | 浅紫 | `purple-400` (#C084FC) |
| 文字主色 | 深灰 | `gray-800` (#1F2937) |
| 文字辅色 | 中灰 | `gray-500` (#6B7280) |
| 边框 | 浅灰 | `gray-200` (#E5E7EB) |
| 成功/赞 | 粉色爱心 | `pink-500` |
| 收藏 | 黄色星星 | `yellow-500` |

## 布局规范

- **最大内容宽度**：`max-w-6xl` (1280px)，居中
- **卡片圆角**：`rounded-xl` (12px)
- **卡片阴影**：`shadow-sm` / hover 时 `shadow-md`
- **间距**：宽松留白，section 之间 `py-8` ~ `py-12`
- **响应式**：移动端 1 列，平板 2 列，桌面 3 列网格

## 组件规范

### 导航栏
- 顶部固定，白色背景 + 底部边框
- 左侧：Logo「天堂次元港」+ 导航链接（首页、搜索）
- 右侧：登录/注册按钮（未登录）或 头像+发帖按钮（已登录）
- 高度：64px

### 帖子卡片
- 圆角卡片，hover 时微微上浮
- 顶部：缩略图（16:9 或 4:3 裁切）
- 底部：标题（1-2 行截断）、标签、点赞数、用户名
- 标签：小圆角 pill 样式，浅蓝背景

### 按钮
- 主按钮：蓝色背景 + 白色文字，圆角 `rounded-lg`
- 次按钮：白色背景 + 蓝色边框 + 蓝色文字
- 危险按钮：红色背景
- 点赞按钮：心形图标，未赞灰色空心，已赞粉色实心

### 表单
- 输入框：浅灰边框，focus 时蓝色边框
- 圆角 `rounded-lg`，高度 44px
- 标签文字：灰色，14px

## 字体

- 正文：系统默认中文字体（苹方 / 微软雅黑 / Noto Sans SC）
- 标题：可考虑导入「ZCOOL KuaiLe」（开源可爱风中文字体，Google Fonts）

## 图标

使用 `lucide-react`：
- 首页：`Home`
- 搜索：`Search`
- 发帖：`PlusCircle`
- 点赞：`Heart`
- 收藏：`Bookmark`
- 评论：`MessageCircle`
- 用户：`User`
- 设置：`Settings`
- 登出：`LogOut`
- 关注：`UserPlus`
